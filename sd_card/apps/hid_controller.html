Important: Firmware Flashing

USB HID mode can interfere with firmware uploads. To flash new firmware:

  1. Hold the BUTTON during boot for 3 seconds (LED will blink green/purple)
  2. OR set USB_HID_ENABLED = false in src/config.h
  3. OR use GPIO0=LOW during power on (hardware upload mode)

Safe Mode: When button is held, USB HID is disabled for that session only.
